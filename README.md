# Smart Thermostat Setup Guide for Humid Climates

Everything you need to know about choosing, installing, and configuring smart thermostats in high-humidity environments like South Florida.

## Why This Guide Exists

Smart thermostats save 10-15% on cooling costs — but only when configured correctly for your climate. The default settings on Nest, Ecobee, and Honeywell thermostats are optimized for moderate climates, not South Florida's extreme heat and humidity. This guide covers the adjustments that make the difference.

## Critical Configuration for South Florida

### Humidity Settings
- **Target humidity**: 45-55% (default is often 60% — too high for mold prevention)
- **Enable overcool mode**: Runs AC 2°F below setpoint to extract more moisture
- **Dehumidify before cooling**: Configure AC to prioritize dehumidification in shoulder seasons

### Schedule Optimization
- **Pre-cool before peak hours** (2-6 PM): Start cooling at 1 PM when electricity is cheaper
- **Night setback**: Only 2-3°F in South Florida (unlike 8-10°F in dry climates) — humidity rises too fast
- **Away mode**: Set to 82°F max (prevents humidity spike that causes mold growth)

### Wiring Notes for Older Miami Homes
Most pre-1990 Miami-Dade homes lack a C-wire. Solutions:
1. Use the Ecobee Power Extender Kit (included)
2. Repurpose the G-wire (fan wire) — only works with Nest
3. Install a 24V transformer add-a-wire kit ($15-30)
4. Run new thermostat wire (professional installation recommended)
